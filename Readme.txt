 NetFramework Project:
 
 The NetFramework is a library designed to help implement common network protocols 
 such as SMTP, POP3, FTP... 
  
 
 
 Requires Java 1.5 or greater
 Dependencies:  
 	Compile: 
 			BjlCore
 			BjlIo
 			AbsoluteLayout
 			swing-layout 1.0.2 or greater
 	
 	Runtime: 
 		Required:
 			BjlCore	
 			AbsoluteLayout (only required if the runtime executes the GUI to browse the file system)
 			swing-layout 1.0.2 or greater (only required if the runtime executes the GUI  to browse the file system)
 		Optional:
 			log4j
